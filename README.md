# 🔐 TLKeys Frontend – Locksmith Service Platform (Nuxt.js)

This is the **frontend** of a locksmith service platform built for a UAE-based company. The application is developed using **Nuxt.js 2** to leverage server-side rendering (SSR), improving both **SEO** and **performance** for end users.

## 🌐 Overview

The platform allows customers to easily:
- Book locksmith services online
- Browse services in a responsive and fast interface
- Benefit from SEO-optimized, scalable pages built with Nuxt.js

It’s designed to offer a seamless and user-friendly experience across devices.

## 🚀 Technologies Used

- **Nuxt.js 2** – Server-side rendering and static generation
- **Vue.js** – Reactive frontend framework
- **Custom API Integration** – Secure connection with backend services
- **SCSS / CSS3** – Styling and responsive design
- **Axios** – API communication

## ✨ Features

- ✅ Server-Side Rendering (SSR) for SEO enhancement
- ✅ Fully responsive design
- ✅ Optimized performance and load speed
- ✅ User-friendly service booking interface
- ✅ Scalable architecture for future expansion
